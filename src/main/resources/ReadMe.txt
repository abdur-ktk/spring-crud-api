1) Download and Install JDK 1.8
2) Download and Install latest Mongodb : https://www.mongodb.com/try/download/community
3) Download and Install latest Maven : https://maven.apache.org/download.cgi

4) Download and Extract STS IDE : https://spring.io/tools
5) Now Unzip Developed project source code and import in your STS IDE.
6) Download and Install MongoDBCompass to access Mongodb 
7) Create a Database on Mongodb : name : db_todo
8) Now Run your application.
9) Now Open following url in your web browser: 

	User Interface : http://localhost:8080/
